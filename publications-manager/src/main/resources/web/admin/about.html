This module allows automatic extraction information regarding authors publication, and update this data to marmotta platform.

The data associated with the publications of the authors are extracted from different sources of scientific information (providers). Each provider has different access methods and formats that are transparent thanks to the internal REDI processes. To extract the information from each of the available providers and check their status you can go to the "Providers Management" section.

Once the data have been collected, a validation process must be carried out prior to assigning the publications to the authors. This is done in the "Data From Providers" section.

Subsequently, in order to access the data in the REDI interface, the central database must be generated. You can do this through the "Data From Providers" interface.

Some of the providers employed require API keys for their operation. These can be configured via the "Configuration" section.
