UPDATE AUTHORS DATA

REPOSITORIO SEMÁNTICO DE INVESTIGADORES DEL ECUADOR

This module allows automatic extraction information regarding authors, and update this data to marmotta platform.

In order to carry out the extraction process of researchers, it is necessary to register the information of the institutions to which the researcher belongs. Organization information can be managed from the "Organization" interface.

Additionally, it is necessary to register the data access points ("endpoints") available to each institution in which the associated authors are located. This can be configured through the "Endpoint" interface.

The processes used during author extraction tasks require integration with components of the LOD-GF platform. The "Configuration" section is available to configure these external dependencies.
