Descripción de REDI

El REDI (Repositorio Ecuatoriano de Investigadores) es un sistema que permite identificar áreas comunes de investigación entre una lista de autores y publicaciones de las diferentes universidades del Ecuador. Para lograr este objetivo, se ha definido una serie de paquetes que permitan:

  • La extracción de los autores por la invocación automática de los servicios que utilizan el estándar OAI-PMH1
  • La extracción de las publicaciones de los autores de diferentes fuentes de datos
  • La descripción semántica de los autores y sus publicaciones con respecto a modelos ontológicos
  • La aplicación de técnicas de minería de datos (clustering) para detectar áreas comunes a través de las palabras clave, título y resumen presentes en las publicaciones.

En este sistema combinamos las ideas de varias iniciativas, y proponemos un nuevo sistema centrado en la identificación de áreas de investigación comunes.

Funcionalidad


Entre las funcionalidades que REDI ofrece se encuentran:

  • Búsqueda de autores
  • Navegación a través de las publicaciones del autor
  • Soporte para consultar publicaciones de autores externos
  • TAG CLOUD: Nube de etiquetas de áreas de investigación. El tamaño de cada nodo depende del número de publicaciones
  • GROUP BY: Se muestran las publicaciones según la área de investigación seleccionada y se agrupan por autores
